How to Record Stats

  1. Select Batter: Choose player from batting lineup
  2. Set At-Bat #: Enter the at-bat number (doesn't auto-increment)
  3. Choose RBI: Select 0-4 runs this batter drove in
  4. Did Player Score: Yes if this batter scored a run
  5. Record Result: Click the at-bat result button
  6. Fix Mistakes: Re-select same at-bat # to correct

Tip: RBI = runs driven in by this batter. Runs = this batter scoring.
